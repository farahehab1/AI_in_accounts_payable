Any additional dependencies or requirements needed to run the code should be explained in the readme file.

The BPMN code , must be run in camunda

Artificial Intelligence in Accounts Payable focuses on improving the efficiency, accuracy, and automation of accounts payable processes through the use of AI technologies such as Machine Learning (ML), Natural Language Processing (NLP), and Robotic Process Automation (RPA). The system is designed to reduce manual effort, minimize errors, improve payment accuracy and support faster financial workflows while maintaining human oversight for approvals and exception handling. The features in my prototype is focused on automating key accounts payable activities including: Goods Receiving Confirmation , Invoice Data Extraction , Invoice Data Matching , Payment Processing and its Bank Verification, Exporting Suppliers Reports and Accounting Records Updating.

For each code to be run it must be run on different platform:

The BPMN: Use https://camunda.com/ for it to give you the right diagram.
